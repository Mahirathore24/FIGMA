# FIGMA

Responsive frontend layout built from the provided Figma reference. Includes a hero slider and product grid matching the desktop layout, with responsive behavior for tablet and mobile.

## Features
- Full-width hero slider with arrows, dots, and autoplay
- Ecommerce-style hero content block
- Product grid: 4 cards per row on desktop, 2 per row on tablet/mobile
- Accessible buttons and semantic HTML structure

## Tech Stack
- HTML5
- CSS3
- JavaScript (ES6+)
- Swiper.js (CDN)

## How To Run
Open index.html in your browser.

## Project Structure
- index.html: main markup
- styles.css: layout and styling
- main.js: slider and interaction logic
